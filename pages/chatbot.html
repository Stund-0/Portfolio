WhatsApp Chatbot 2026

A virtual assistant designed to automate customer support and medical appointment scheduling through WhatsApp. Built with Python, Flask, and PostgreSQL, the system interprets user requests, answers frequently asked questions, and schedules appointments automatically using the Meta WhatsApp Cloud API.

Key Features

  • Detects over 20 user intents using rule-based logic and regular expressions.
  • Schedules medical appointments by collecting name, phone number, specialty, date, and time.
  • Maintains multi-turn conversation context for each user.
  • Sends appointment approval or rejection requests to administrators through WhatsApp.
  • Automatic notifications for new appointments, human handoff requests, and after-hours messages.

A project developed entirely by me.
